uBlacklist is a browser extension that filters Google Search results, available for Chrome, Firefox, and Safari.

- [Chrome Web Store](https://chrome.google.com/webstore/detail/ublacklist/pncfbmialoiaghdehhbnbhkkgmjanfhe/)
- [Firefox Add-ons](https://addons.mozilla.org/en/firefox/addon/ublacklist/)
- [App Store](https://apps.apple.com/us/app/ublacklist-for-safari/id1547912640) (maintained by [Group-Leafy](https://github.com/HoneyLuka/uBlacklist/tree/safari-port/safari-project))
- [GitHub](https://github.com/iorate/ublacklist)

## Features {#features}

- Completely block unwanted sites from appearing in search results
- Block sites by specific URLs using match patterns or regular expressions
- Block sites by specific titles using regular expressions
- Highlight preferred sites in search results
- Support for Bing, Brave, DuckDuckGo, Ecosia, SearXNG, Startpage.com, Yahoo! JAPAN, and Yandex
- Synchronize rulesets across devices using Google Drive or Dropbox
- Subscribe to public rulesets
